# TASK
Write a simple function that checks whether a string is a mirror word. 

A mirror word is a string that remains identical when all non-alphabetic 
characters are ignored, and capitalization is normalized.

## Requirements
- Your function should return a boolean (true or false) indicating whether the string is a mirror word.
- Ignore all characters except for alphabetic letters (a-z, A-Z).
- Treat uppercase and lowercase letters as equivalent.

## Example Inputs and Outputs
"A man, a plan, a canal, Panama" → true
"Race a car" → false
"No lemon, no melon!" → true
